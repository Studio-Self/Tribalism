# Common Room Integration Guide

## Overview
Common Room is a community analytics platform that unifies member data across multiple platforms (Discord, Slack, GitHub, Twitter, etc.) into a single view.

## Use Cases
- Track member activity across platforms
- Identify your most active and valuable members
- Measure community health metrics over time
- Segment members for targeted engagement
- Generate reports for stakeholders

## Setup
1. Connect your community platforms (Discord, Slack, GitHub, etc.)
2. Connect social platforms (Twitter, LinkedIn)
3. Set up member identity resolution (merge duplicate profiles)
4. Configure segments and alerts

## Key Features
- Unified member profiles across platforms
- Activity timelines per member
- Automated segments and alerts
- Custom reports and dashboards
- Integration with CRM (Salesforce, HubSpot)

- Set up "at-risk" alerts for previously active members going quiet
- Create segments for ambassador candidates (high activity + long tenure)
