# Luma Integration Guide

## Overview
Luma is an event management platform designed for community events. Handles registration, reminders, check-ins, and event discovery.

## Use Cases
- Community AMAs, workshops, and meetups
- Event registration and attendance tracking
- Building an event calendar for your community
- Growing through Luma's event discovery

## Setup
1. Create a Luma calendar for your community
2. Set up your community profile
3. Create events with clear descriptions and speaker info
4. Embed event links in your community platform

## Key Features
- Beautiful event pages
- Automated email reminders
- Attendee management
- Analytics (registration, attendance, engagement)
- Calendar subscriptions
- Event discovery (helps with growth)

## Best Practices
- Use the Luma calendar as your community's event hub
- Collect emails through registration for community growth
- Follow up with attendees post-event (convert to community members)
- Use recurring events for regular programming
