# Stripe Integration Guide

## Overview
Stripe handles payment processing for paid communities, memberships, and event ticketing.

## Use Cases
- Paid community membership billing
- Event ticket sales
- Ambassador/affiliate commission payouts
- Tiered pricing and upgrades

## Setup
1. Create a Stripe account
2. Set up products and pricing (monthly/annual subscriptions)
3. Connect to your community platform (Circle, Memberful, custom)
4. Configure webhooks for member provisioning

## Key Features
- Subscription management
- Customer portal (members manage their own billing)
- Revenue analytics
- Coupon and promotion codes
- Multiple currency support

## Best Practices
- Offer annual pricing at a discount (reduces churn)
- Use Stripe's customer portal so members can self-manage
- Set up dunning emails for failed payments
- Track MRR and churn in Stripe dashboard
- Use promotion codes for founding member discounts
