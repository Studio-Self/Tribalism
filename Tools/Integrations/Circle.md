# Circle Integration Guide

## Overview
Circle is a community-first platform built for creators, course builders, and paid membership communities. It combines discussion spaces, events, courses, and member management in one tool.

## Community Setup

### Spaces
- Spaces are Circle's version of channels/groups
- Can be public, private, or paid
- Support posts, events, and rich media

### Recommended Structure
```
Welcome Space (public)
├── Getting Started
├── Introductions

Core Spaces
├── Main Discussion
├── Topic Spaces (2-3)
├── Events

Premium Spaces (optional, paid tier)
├── Exclusive Content
├── Mastermind Groups
├── Direct Access
```

### Features
- Native live streams and events
- Course builder
- Membership and payment built-in
- Member directory with profiles
- Headless option for embedding in your product

### Pricing
- Basic: $49/month
- Professional: $89/month
- Business: $199/month
- Enterprise: Custom

## Best Practices
- Use Space categories to organize by topic
- Enable member profiles for networking
- Leverage native events for community programming
- Use the digest email feature for re-engagement
- Set up welcome automations for new members
