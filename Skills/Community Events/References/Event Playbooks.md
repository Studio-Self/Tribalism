### Live AMA Facilitation Script

```
[5 min before start]
@here Our AMA with [Guest] starts in 5 minutes!
Get your questions ready. One question per message so people can upvote.

[Start]
Welcome everyone! Today we have [Guest Name] with us.

[Guest bio ‚Äî 3-4 sentences]

[Guest], welcome! Before we jump into questions, could you share
a quick intro and what you're excited to talk about today?

[After guest intro]
Let's get into it. I'll pull from pre-submitted questions first,
then open it up for live questions.

[Moderate: select highest-upvoted questions, tag guest,
keep pace ‚Äî aim for 1 question every 5-7 minutes]

[15 min before end]
We've got time for about 3 more questions! Get them in now.

[End]
That's a wrap! Huge thanks to [Guest] for an incredible AMA.

Key takeaways:
1. [Takeaway]
2. [Takeaway]
3. [Takeaway]

This thread will stay pinned for anyone who missed it.

[Guest], where can people follow your work?
[Let guest share their links]
```

## Workshop Playbook

### Workshop Planning Template

```
Workshop Title: [Title]
Host: [Name]
Date/Time: [Date] at [Time] [Timezone]
Duration: [60-90 minutes]
Max Attendance: [15-30 for interactive]
Format: [Live video + shared doc / Screen share + breakouts]

Learning Objective:
By the end, participants will be able to [specific, actionable skill].

Pre-work (optional):
- [What to read/watch/bring]

Materials Needed:
- [Slides/doc/template link]
```

### Workshop Run of Show

| Time | Segment | What Happens |
|------|---------|-------------|
| 0:00-0:10 | Welcome + Intro | Host intro, workshop overview, set expectations |
| 0:10-0:30 | Teaching | Core content, demonstration, examples |
| 0:30-0:50 | Hands-On | Participants practice, work on exercise |
| 0:50-1:05 | Share + Discuss | 3-5 participants share their work, group feedback |
| 1:05-1:15 | Q&A + Wrap | Open questions, key takeaways, next steps |
| 1:15-1:20 | Close | Thank participants, share recording info, CTA |

## Event Benchmarks

| Metric | Virtual Events | In-Person Events |
|--------|---------------|-----------------|
| Registration to attendance | 35-50% | 60-80% |
| Average duration (engagement drop-off) | 45-60 min optimal | 2-3 hours |
| Chat engagement (msgs per attendee) | 2-5 | N/A |
| Post-event survey response | 15-30% | 30-50% |
| Event ‚Üí community join rate | 10-25% (non-members) | 20-40% |
| Repeat attendance rate | 30-50% | 40-60% |
| Optimal frequency | Weekly or bi-weekly | Monthly or quarterly |
| Min viable attendance | 8-10 for interactive | 15-20 |
